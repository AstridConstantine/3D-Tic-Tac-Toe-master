# 3D-Tic-Tac-Toe

Original Source
https://github.com/adamdevigili/3D-Tic-Tac-Toe/tree/master

Compilation/Execution
------------
```$ cd /whatever/folder/3D-Tic-Tac-Toe/src```

```$ javac TTT3D.java```

```$ java TTT3D```


Simulations
------------
```$ cd /whatever/folder/3D-Tic-Tac-Toe/src```

```$ javac TTT3D.java SimulationRunner.java```

```$ java SimulationRunner```
